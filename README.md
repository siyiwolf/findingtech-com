## 📋 项目概览

本项目包含Finding公司智能硬件业务方向的完整市场洞察、产品规划、GTM策略和项目管理计划。涵盖两个核心方向：

1. **露营装备** - 智能温控帐篷、太阳能电源、智能露营生态系统
2. **机器人集群** - Finding Pro系列、CampBot营地系统、仓储分拣机器人、多机器人编队

## 📄 报告说明

每个洞察报告包含8个模块：
1. 市场机会概述
2. 用户画像与核心痛点
3. 竞争格局分析
4. 产品概念与差异化定位
5. GTM策略
6. 实施路线图
7. 风险评估与应对
8. 数据来源

## 🛠️ 工具使用

### HTML转PDF
```bash
# 使用Chrome无头模式
tools/html-to-pdf.sh <input.html> <output.pdf>

# 或手动
google-chrome --headless --disable-gpu --no-sandbox \
  --print-to-pdf=output.pdf input.html
```

### 查看报告
直接用浏览器打开HTML文件，或使用生成的PDF文件。
