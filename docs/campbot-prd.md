## 1. 产品概述

### 1.1 产品定义

CampBot 是一套全自动智能营地搭建与管理系统，包含6台专用机器人 + 中央控制站（ControlHub），实现从开箱到营地完全搭建 ≤ 20分钟，减少80%人力需求。

### 1.2 目标市场

- **高端度假村**：提升服务效率，降低人力成本
- **豪华露营地（Glamping）**：高科技体验，差异化竞争
- **企业团建基地**：标准化、可重复的活动执行
- **军事/应急训练营**：快速部署，安全可靠

### 1.3 核心价值主张

| 价值点 | 说明 | 量化指标 |
|--------|------|----------|
| 效率提升 | 自动搭建，无需人力 | 20分钟 vs 60-120分钟 |
| 成本节约 | 减少10人团队至2人监督 | 人力成本节约 ¥730k/年 |
| 全天候作业 | 雨天/夜间照样工作 | 无天气限制 |
| 安全零事故 | 自动化避免人工风险 | 工伤率 = 0 |
| 质量标准化 | 每次搭建完全一致 | 客户满意度 > 95% |

### 1.4 定价

- 标准版：¥299,000/套（6机器人 + ControlHub）
- 定制版：¥399,000（品牌LOGO、定制固件）
- 租赁：¥15,000/月（含基础维护）
- 年服务套餐：¥29,000/年（7x24支持、保养、升级）

## 3. 功能规格

### 3.1 AssemblyBot（搭建机器人）

**功能**：
- 自动展开帐篷框架
- 使用冲击扳手打地钉（扭矩可调）
- 视觉定位（Intel RealSense D455）
- 与CarrierBot协同（等待配件到位）

**性能**：
- 负载：2kg（工具）
- 精度：±0.1mm
- 速度：搭建3人帐篷 ≤ 15分钟

---

### 3.2 CarrierBot（搬运机器人，×2）

**功能**：
- 运输帐篷主体、配件（地钉、工具）
- AGV导航（激光SLAM）
- 自动装卸货（机械爪）

**性能**：
- 载重：50kg
- 速度：1.5 m/s
- 续航：8小时

---

### 3.3 GuardBot（安全巡逻）

**功能**：
- 营地周界巡逻（预设路径）
- 环境监测：CO、烟雾、温湿度
- 异常检测（AI识别）
- SOS告警（通过ControlHub上报）

**性能**：
- 传感器：多气体 + 摄像头
- 巡逻频率：每30分钟一圈
- 覆盖范围：200m x 200m

---

### 3.4 PowerBot（能源管理）

**功能**：
- 太阳能板自动部署与角度调节（MPPT）
- 电池管理（24V 10Ah 磷酸铁锂）
- 为全系统供电（6机器人 + ControlHub）
- 充电策略：优先保障关键设备

**性能**：
- 太阳能板：200W × 2
- 充电时间：4小时（晴天）
- 续航支持：48小时（无日照）

---

### 3.5 CookBot（烹饪辅助）

**功能**：
- 协作机械臂操作炊具（炉灶、锅具）
- 温度监控（K型热电偶）
- 安全防护（火焰探测、油温超限自动关闭）

**性能**：
- 负载：3kg（锅具）
- 温度范围：0-300°C
- 安全性：ISO 10218

---

### 3.6 ClimateBot（环境调节）

**功能**：
- 半导体片制冷/加热（TEC1-12706 × 4）
- 温湿度传感器闭环控制（SHT30）
- 导风罩定向送风
- 与帐篷集成（内衬通风管道）

**性能**：
- 制冷量：200W
- 加热量：300W
- 控温范围：15°C - 30°C
- 噪音：< 40dB

---

### 3.7 ControlHub（中央控制）

**硬件**：
- NVIDIA Jetson Orin (8-core ARM, 32GB RAM)
- 5G/Wi-Fi 6 通信
- UPS 备用电源（30分钟）

**软件**：
- ROS 2 Humble + Navigation2
- 自定义调度算法（拍卖算法）
- Web管理界面（HTTP + WebSocket）
- 设备管理、任务编排、实时监控

## 4. 技术架构

### 4.1 软件栈

```
┌─────────────────────────────────────────────┐
│              Application Layer              │
│   CampBot App (iOS/Android) - 蓝牙/Wi-Fi   │
│   Web Console (可选) - 远程监控            │
├─────────────────────────────────────────────┤
│           ControlHub (Jetson Orin)          │
│   ROS 2 Core + Swarm Scheduler             │
│   Navigation2 (AMCL + DWB)                 │
│   Behavior Trees (BT)                      │
├─────────────────────────────────────────────┤
│           Device Layer (6 Robots)           │
│   ROS 2 Nodes (各自独立) + Firmware        │
│   Motor Control, Sensor Fusion             │
└─────────────────────────────────────────────┘
```

### 4.2 通信协议

- **机器人 ↔ ControlHub**：ROS 2 (DDS over UDP)，10Hz 状态发布
- **ControlHub ↔ Cloud**：gRPC + TLS（设备管理、日志）
- **App ↔ ControlHub**：HTTP REST + WebSocket（本地直连）

### 4.3 安全机制

- 设备认证：X.509证书（每个机器人唯一）
- 急停：硬件急停按钮 + 软件急停广播
- 地理围栏：限制机器人活动范围（帐篷周围10m）
- 日志审计：所有操作不可篡改

### 5.1 购买阶段

客户咨询（展会/官网） → 需求沟通（销售） → 方案报价 → 合同签署 → 付款

---

### 5.2 交付与部署

```
物流发货 → 现场接收 → 开箱验货 → App配置 → 自动校准 → 首次搭建演示
```

**时间**：1天（现场安装培训）

**交付物**：
- 6台机器人 + ControlHub + 帐篷套装
- 快速入门指南
- 2小时培训（操作+维护）

---

### 5.3 使用阶段

```
日常使用：
1. 客户App选择帐篷类型，点击"开始搭建"
2. 机器人唤醒，协同完成搭建（20分钟）
3. GuardBot开始巡逻，ClimateBot监测环境
4. 客户通过App查看实时状态

拆收：
1. 点击"拆收"指令
2. 机器人自动拆卸、打包（10分钟）
3. 充电底座待机
```

**支持**：
- 7x24远程支持（电话/App内客服）
- 定期保养（每月1次，服务套餐包含）
- 固件OTA升级（季度更新）

---

### 5.4 维护与升级

**预防性维护**：
- 每月：检查机械臂润滑、传感器校准
- 每季度：电池健康检测、固件升级
- 每年：全面检修 + 关键部件更换

**升级路径**：
- v1.1：ClimateBot量产（2026 Q1）
- v2.0：AI调度算法（协同优化25%）

## 8. 用户界面要求

### 8.1 CampBot App

**主要屏幕**：
1. **主页**：一键"开始搭建" / "拆收" 按钮
2. **状态页**：各机器人实时状态（电池、位置、任务）
3. **地图页**：营地平面图 + 机器人位置可视化
4. **设置页**：帐篷类型选择、定时任务、历史记录

**设计原则**：
- 极简：主要操作3步内完成
- 状态一目了然（颜色编码：绿色/黄色/红色）
- 离线可用（搭建过程不依赖网络）

---

### 8.2 Web控制台（可选，企业客户）

**功能**：
- 设备管理（注册、状态、OTA）
- 任务队列（手动创建任务）
- 数据分析（搭建次数、时长、故障统计）
- 用户管理（多租户）
