## 2. 目标客户画像

### 2.1 B端客户（60%收入占比）
**核心目标：**
- **高端露营基地**（200+家）：莫干山、安吉、崇礼等头部营地
- **户外装备经销商**（50+家）：迪卡侬、三夫户外、牧高笛渠道商
- **企业团建服务商**（30+家）：阿里、腾讯、字节跳动企业团建供应商

**痛点：**
- 客户投诉"睡不好"是露营差评主因（占38%）
- 装备采购成本高，希望溢价服务提升客单价
- 差异化竞争需求，智能装备是亮点

### 2.2 C端客户（40%收入占比）
**核心目标：**
- **高净值户外爱好者**（年收入30万+，年龄段28-45岁）
- **科技极客**：智能家居用户，喜欢尝试新科技
- **家庭露营用户**：有孩子的家庭，注重舒适性和安全性

**用户画象：**
- **典型用户**：张伟，35岁，互联网公司产品经理，年收入40万，已购买智能家居设备，去年露营8次
- **购买动机**：孩子2岁，希望露营时有家里一样的睡眠体验；愿意为好睡眠支付溢价
