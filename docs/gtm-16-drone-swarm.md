## 1. 产品概述

### 1.1 产品组合

| 产品编号 | 产品名称 | 核心功能 | 定价 |
|----------|----------|----------|------|
| 16-Basic | 无人机集群基础版 | 10架标准机 + SwarmHub + 基础软件 | ¥800,000 |
| 16-Pro | 无人机集群企业版 | 25架 + 高级算法 + 5G通信 + 2年维护 | ¥2,000,000 |
| 16-Ultra | 无人机集群定制版 | 按需规模 + 特殊负载 + 私有化部署 | ¥5,000,000+ |
| **服务套餐** | **年度维护** | 7x24支持、软件升级、定期演练 | ¥200,000/年 |

### 1.2 核心价值主张

- **效率提升**：多机协同，作业效率相比单机提升5-10倍
- **成本降低**：人力成本降低70%，单机成本降低30%
- **安全性**：全自动飞行，零伤亡事故（对比人工高空作业）
- **可扩展性**：模块化集群，5-50架弹性扩容
- **智能化**：AI任务规划与自主避障，路径优化提升20%

## 2. 目标客户画像

### 2.1 B端客户（主要收入）

**核心目标：**

| 客户类型 | 数量预估 | 应用场景 | 价格敏感度 |
|----------|----------|----------|------------|
| 露营/营地运营商 | 200+家 | 巡检、表演、引导 | 中 |
| 景区管理方 (5A/4A) | 300+家 | 导览、表演、监控 | 高 |

## 3. 定价策略

### 3.1 价格锚定

| 维度 | 单机方案 (10架独立) | 集群方案 (16-Basic) |
|------|-------------------|-------------------|
| 硬件成本 | ¥250,000 (10x25k) | ¥400,000 (含SwarmHub/中继) |
| 软件成本 | 0 (无集群功能) | ¥100,000 |
| 部署/培训 | ¥50,000 | ¥100,000 |
| 总成本 | ~¥300,000 | ~¥600,000 |
| **售价** | ¥350,000+ | **¥800,000** |
| 功能差异 | 单机独立，无协同 | 集群控制 + 协同 + 统一管理 |

**ROI计算：**
- 客户使用单机：10人团队，每人操作1架，月人力成本¥50,000
- 使用集群：2人操作，人力成本¥10,000，节省¥40,000/月
- 投资回报期：¥800k - ¥350k = ¥450k / ¥40k = **11.25个月**

### 3.2 定价组合

**分层定价：**
- **Basic**：¥800,000 - 入门，10架，适合小型营地/景区
- **Pro**：¥2,000,000 - 企业级，25架+5G+高级算法，适合大型巡检/景区
- **Ultra**：¥5,000,000+ - 定制化，50架以上，特殊负载，私有化部署

**服务套餐：**
- 年度维护：¥200,000/年（7x24支持、软件升级、季度演练）
- 培训套餐：¥100,000（5天现场+2天高级编队训练）
- 保险套餐：¥50,000/年（第三方责任险、设备损失险）

**租赁选项：**
- 月租：¥100,000/月（含10架 + SwarmHub + 基础支持）
- 年租：¥800,000/年（节省2个月租金）

## 4. 渠道策略（混合直销+渠道）

### 4.1 直销团队（50%）

**目标客户**：大型项目（>¥1M），战略客户
- 工业检测公司（国家电网、中石油管道局）
- 大型景区集团（华侨城、长隆、迪士尼）
- 应急救援总队（省市级）

**销售流程：**
1. 线索获取（展会、官网、KOL推荐）
2. 初步沟通（电话/线上会议，需求诊断）
3. 方案设计与POC（2-4周，客户现场演示10架基础版）
4. 商务谈判（合同、付款条款、交付周期）
5. 签约与交付（4-8周）
6. 售后跟进（服务套餐续约）

**团队配置：**
- 销售总监 x1 (覆盖北上广深)
- 销售经理 x3 (华东、华南、华西)
- 售前工程师 x2 (POC演示、技术答疑)

### 4.2 渠道代理（30%）

**代理商类型：**
- **无人机渠道商**：已有大疆、Skydio代理关系的，叠加集群产品
  - 佣金：15-20%
  - 要求：有一定技术能力，可做POC
- **系统集成商 (SI)**：为景区、工厂提供整体解决方案
  - 佣金：10-15%
  - 要求：项目经验丰富，客户关系强
- **设备租赁公司**：向中小客户提供租赁服务
  - 佣金：10%
  - 要求：资金实力强，可采购设备库存

**渠道政策：**
- 独家区域代理（省级）：佣金 20%，销量要求 5套/年
- 非独家：佣金 15%
- 项目报备制：30天内有效，避免渠道冲突

### 4.3 线上渠道（20%）

**官网(DTC)：**
- 产品展示 + 参数配置器 + 在线询价
- 内容营销：案例库、白皮书、视频演示
- 线索收集：免费POC申请表单

**电商平台：**
- 阿里巴巴1688（B2B）：产品上架，吸引小批量采购
- 京东企业购：大型企业采购平台入驻
- 不直接销售（高单价），仅展示+留资

## 6. 销售流程

### 6.1 客户旅程

```
认知 → 兴趣 → 考虑 → POC → 购买 → 使用 → 忠诚

阶段：认知

- 指标：POC成功率、客户反馈评分

- 商务谈判（价格、交付、服务）
- 合同签署
- 指标：销售周期、成交率、平均订单金额

阶段：使用
- 部署、培训、上线
- 售后服务
- 指标：客户满意度 (CSAT)、NPS

阶段：忠诚
- 服务套餐续约
- 升级/扩容
- 推荐新客户
- 指标：续约率、客户生命周期价值 (LTV)
```

## 7. 竞争分析

### 7.1 主要竞争对手

| 竞品 | 类型 | 优势 | 劣势 | 我们的对策 |
|------|------|------|------|-----------|
| 大疆 (DJI) Matrice + 机巢 | 单机+自动起降 | 品牌强、渠道广、可靠性高 | 无集群协同功能，多机任务不协调 | 强调集群价值，展示多机编队优势 |
| Skydio X + Dock | 自主飞行+机巢 | 美国品牌，AI避障强 | 价格高、国内受限、集群能力弱 | 本土化支持+集群算法领先 |
| 禾赛科技等激光雷达方案 | 硬件传感器 | 技术垂直深入 | 非完整解决方案 | 提供端到端系统，降低集成难度 |
| 自研系统（客户内部） | 定制开发 | 完全适配 | 周期长、成本高、无商业支持 | 快速交付+持续更新+生态 |

### 7.2 竞争优势

1. **算法领先**：集群协同算法经过多年研发，支持50架稳定运行
2. **通信可靠**：5G + LoRa 自组网冗余，适应无网环境
3. **本土化**：中文UI、北斗服务、国内售后、政策合规支持
4. **成本优势**：自研飞控，硬件BOM更低
5. **快速交付**：标准产品4-8周交付，定制12周
