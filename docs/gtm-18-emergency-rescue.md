## 1. 产品组合与定价

### 1.1 产品定义
**核心产品线**：
- **标准版（个人）**：基础定位 + 北斗短报文SOS + 急救包 + 10,000mAh电源
- **Pro版（家庭）**：标准版 + 语音急救指导 + 蓝牙App + 12,000mAh电池
- **专业版（救援队）**：Pro版 + 对讲模块 + 多设备协同 + 太阳能板

### 1.2 定价策略
| 版本 | 价格 (RMB) | 毛利率 | 目标客户 |
|------|------------|--------|----------|
| 标准版 | ¥2,499 | 45% | 户外爱好者、家庭 |
| Pro版 | ¥3,999 | 50% | 重度户外、家庭应急 |
| 专业版 | ¥8,888 | 55% | 救援队、企业/机构 |

**促销策略**：
- 首发优惠：早鸟价 ¥2,199（标准版）/ ¥3,599（Pro）
- 捆绑销售：家庭两件装 ¥6,999（Pro×2）
- B2B报价：10套起 9折，50套起 8折

### 1.3 定价依据
- 参考竞品：卫星电话 ¥5,000-15,000，急救包 ¥200-500，户外电源 ¥1,000-3,000
- 一体化优势：价格低于单品总和（¥8,000+），价值感知高
- ROI：获救价值远高于售价，心理门槛低

## 2. 渠道策略

### 2.1 渠道分类
- **线上渠道**：
  - DTC官网 + 小程序商城
  - 天猫/京东旗舰店
  - 抖音/快手直播带货
- **线下渠道**：
  - 户外用品店（迪卡侬、探路者）
  - 消防/应急用品专卖店
  - 展会（中国国际应急管理展）
- **B2B渠道**：
  - 政府应急管理部门（公开招标）
  - 企业采购（EHS部门）
  - 学校/培训机构（安全演练配套）

### 2.2 渠道选择依据
- 目标用户获取成本：线上低，线下体验好
- 初期聚焦线上DTC+天猫，控制毛利
- 中后期拓展B2G/B2B，批量订单

### 2.3 渠道政策
- 线上：一件代发，7天无理由退换（未使用）
- 线下：提供体验样机 + 销售返点 15%
- B2B：定制培训 + 三年保修

## 3. 营销计划

### 3.1 品牌传播
- **Slogan**: "一装在手，安全随行"
- **核心信息**: 专业救援级、一键SOS、模块化设计
- **内容营销**:
  - 短视频：用户故事（真实救援案例）
  - 直播：产品功能演示、现场防水测试
  - 图文：专业评测（户外KOL、救援博主）
- **广告投放**:
  - 抖音/小红书信息流（定向户外、旅游、安全标签）
  - 百度SEM（应急装备关键词）
  - 行业媒体（中国应急管理报）

- **种子用户**：免费赠送100套给救援队、户外俱乐部，换取评测报告和推荐。
- **场景体验**：在登山/露营活动中设立体验点，现场演示SOS发送。
- **口碑营销**：老用户推荐返现 ¥200/人。
- **公关活动**：发布《中国户外安全报告》（联合中国登山协会）。

## 4. 销售流程

### 4.1 销售漏斗
```
曝光（广告） → 点击（落地页） → 咨询（客服/在线） → 下单 → 复购/推荐
```

### 4.2 转化策略
- 落地页：突出30秒演示视频 + 用户证言 + 价格锚点（原价¥3,999，现价¥2,199）
- 客服：7×12小时在线，专业产品顾问
- 下单：提供“安心险”（3年内故障免费换新，¥299）
- 复购：配件折扣（电池、急救耗材）

### 4.3 销售工具
- 产品演示视频（功能、使用场景）
- 销售FAQ（应对价格质疑、竞品对比）
- CRM系统：记录用户使用场景，推送耗材提醒

## 5. 竞争应对

### 5.1 竞品分析
| 竞品 | 优势 | 劣势 | 应对 |
|------|------|------|------|
| 传统急救包 | 医疗认证、价格低 | 功能单一 | 强调一体化价值 |
| 卫星电话 | 全球通信成熟 | 价格高、体积大 | 突出性价比和易用性 |
| 户外电源 | 大容量、多接口 | 无通信/定位 | 差异化定位（安全装备） |
| 智能手表 | 日常佩戴、健康监测 | 续航短、户外弱 | 强调专业性和续航 |

### 5.2 应对策略
- **技术壁垒**：申请模块化设计专利、加密通信算法
- **服务优势**：提供应急救援绿色通道（与救援中心合作）
- **价格战略**：定价介于传统急救包和卫星电话之间，突出“一步到位”
- **渠道保护**：与户外连锁店签订排他协议（前6个月）
