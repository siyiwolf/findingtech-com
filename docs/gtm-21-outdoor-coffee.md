## 5. 销售流程

```
线上购买：
客户浏览 → 加入购物车 → 支付 → 发货（顺丰） → 收货 → App 激活 → 售后

B端销售：
线索 → 样品试用 → 批量报价 → 合同 → 交付 → 培训 → 定期回访
```

### 5.1 渠道管理

- **线上直营**：官网 + 小米有品（主要）
- **电商平台**：京东自营、淘宝旗舰店
- **线下体验**：户外用品店（如三夫户外）、咖啡器具店
- **B端渠道**：直销团队（3人）覆盖露营地、民宿、活动公司

### 5.2 促销策略

- 早鸟/众筹：9折
- 捆绑：咖啡机 + 豆子/杯子套装 95 折
- 节日营销：618/双11 满减，Christmas 礼盒
- 以旧换新：旧咖啡设备抵 ¥200

## 6. 竞争应对

### 6.1 竞争者矩阵

| 竞品 | 优势 | 劣势 | CampBrew 对策 |
|------|------|------|---------------|
| Wacaco Minipresso | 品牌知名度高、轻便 | 需另配磨豆、无温控 | 一体化 + 智能温控，功能碾压 |
| 手冲器具套装 | 极致专业感、便宜 | 多件搬运麻烦、操作复杂 | 一键完成，户外省心 |
| 家用全自动咖啡机 | 功能全、品质高 | 不能移动、耗电大 | 便携 + 电池续航，场景互补 |
| 胶囊咖啡机（便携版） | 操作极简 | 胶囊贵、不可自定义 | 支持现磨咖啡豆，口感更好，长期成本低 |

### 6.2 差异化策略

- **场景卡位**：专注"户外+轻咖啡"，不做家用大机型
- **技术壁垒**：PID + 锥刀磨豆 + 自动冲洗组合
- **社群运营**：建立 CampBrew 用户社群，定期举办露营咖啡活动
- **订阅经济**：通过咖啡豆订阅锁定用户 LTV
