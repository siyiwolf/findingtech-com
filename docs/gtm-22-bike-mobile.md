## 2. 目标客户

### 2.1 B端
- 骑行俱乐部（团建、赛事）
- 车店（增值服务、配件捆绑）

### 2.2 C端
- 城市通勤族（10-30km/天）：重视安全和导航
- 运动爱好者（周末训练）：数据训练、视频分享
- 长途骑手（318、青海湖等）：续航、SOS
- 骑行Vlogger：云台拍摄、快速剪辑

## 3. 定价策略

**锚定对比**：
- Garmin 码表 ¥2000-5000（单一功能）
- 小米智能车灯 ¥100-200
- GoPro + 稳定器 ¥3000+

**定价逻辑**：基础版性价比突出（比Garmin便宜，功能更全）；Pro版瞄准高端训练和拍摄人群。

**促销**：
- 早鸟：基础版 ¥1,299，Pro版 ¥2,499
- 套装：+太阳能板 捆绑价 ¥1,799 / ¥3,199
- 自行车展会9折

## 4. 营销计划

### 4.1 核心信息
"骑行一机搞定：导航、灯光、拍摄、训练，所有数据都在掌心"

### 4.2 营销活动
- **安全骑行campaign**：与交警合作宣传夜间安全，赠送装备给骑行志愿者
- **Vlog大赛**：举办"我的骑行故事"视频大赛，奖品为RideHub Pro
- **KOL合作**：骑行UP主（如"小曼骑行记"）、运动装备测评博主
- **车店体验**：在全国200家车店设置体验台，扫码试用

### 4.3 销售渠道
- 线上：官网 + 京东 + 小米有品
- 线下：捷安特/美利达门店 + 三夫户外 + 骑行展会
- B端直销：俱乐部、赛事组织、企业团建

## 5. 竞争应对

| 竞品 | 劣势 | RideHub优势 |
|------|------|-------------|
| Garmin码表 | 贵、功能单一 | 价格一半，多功能集成 |
| 手机 + 支架 | 不安全、续航差、不防水 | 专用设备，安全可靠 |
| GoPro + 稳定器 | 操作复杂、需多设备 | 一体化语音控制 |
| 传统车灯 | 只是照明 | 智能感应 + 刹车提示 + SOS |

**打法**：教育市场"一机替代多设备"，轻量化出行。
