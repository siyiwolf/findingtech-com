## 4. 销售流程

```
线索 → 筛选 → 初步演示 → 深度试用 → 方案报价 → 合同签订 → 交付验收
```

### 4.1 流程详解
1. **线索获取**: 展会、官网表单、代理商
2. **筛选**: BANT 预算/ authority/ need/ timeline → 合格线索转销售
3. **初步演示**: 在线视频会议或现场Demo (1小时)
4. **深度试用**: 7天免费试用 (需押金，国内客户)
5. **方案报价**: 根据配置生成正式报价单 (3工作日)
6. **合同签订**: 标准合同 + SLA (服务水平协议)
7. **交付验收**: 现场安装培训，客户签字验收

### 4.2 销售周期
- 平均: 45-60 天
- 试用期: 7 天
- 目标转化率: 线索→成交 15%

### 5.1 主要竞争者
| 竞争者 | 类型 | 劣势 | CampBot 对策 |
|--------|------|------|-------------|
| Boston Dynamics Spot | 通用机器人 | 价格$70万+、无帐篷专用功能 | 专用设计，¥30万/套，价格优势60%+ |

| Coleman 传统装备 | 帐篷/睡袋 | 无自动化 | 突出效率提升 (20分钟 vs 2小时) |
| 国内初创 (如逸仙) | 细分场景 | 规模小、服务网络弱 | 全国服务网点 (8个) |

### 5.2 竞争策略
- **差异化**: 专用 vs 通用，全系统集成
- **价格**: 渗透定价，快速占领市场
- **服务**: 24/7 客服，2小时响应
- **生态**: 与帐篷厂商战略合作，打包销售
