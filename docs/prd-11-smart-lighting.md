## 1. 产品概述

### 1.1 产品名称
**产品代号：** SmartLight Pro (SLP-200)  
**产品定位：** 户外场景专用智能照明系统，兼顾庭院、露营、商业氛围

### 1.2 产品简述
SmartLight Pro 是一款集成了高亮度LED、智能控制、无线供电的户外智能照明系统。支持RGB色彩调节、亮度无级调光、情景模式预设。适用于庭院装饰、露营营地照明、商业户外氛围营造等场景。核心优势：IPX8防水、50小时续航（低亮）、Matter协议直连主流智能生态。

### 1.3 核心价值主张
- **全场景防水：** IPX8等级，可浸水1米，应对暴雨和临时积水
- **超长续航：** 太阳能辅助 + 3000mAh电池，低亮模式续航>100小时
- **生态互联：** 支持Matter协议，一键接入Apple Home/Google Home/Alexa
- **灵活安装：** 磁吸、挂扣、螺丝 triplet 安装，无需布线

### 1.4 目标市场
- **B2C：** 庭院家庭、露营爱好者、户外聚会
- **B2B：** 餐厅露台、露营基地、商业街区
- **B2G：** 公园、景区照明升级

## 2. 功能规格

### 2.1 硬件规格

| 组件 | 规格 | 备注 |
|------|------|------|
| LED光源 | COB集成，150lm/W，CRI>80，CCT 2700K-6500K | 支持RGB，1600万色 |
| 光学设计 | 透镜+反射器，配光角120° | 防眩光 |
| 主控MCU | ESP32-C3 (RISC-V, 160MHz) | Wi-Fi 4 + Bluetooth 5.0 |
| 电源管理 | 3000mAh 锂聚合物 + 20W太阳能板（选配） | Type-C快充 (15W) |
| 控制方式 | 物理按键 + 电容触摸 + APP + 语音 | |
| 防护等级 | IPX8 (1米水深，30分钟) | |
| 外壳材质 | PC+GF30% 阻燃，UV稳定 | |
| 安装方式 | 磁吸底座 + 挂扣 + M4螺丝孔 | 三合一 |
| 重量 | ≤150g (主灯) + 太阳能板 180g | |
| 尺寸 | Ф85mm × H45mm (主灯) | |

### 2.2 软件功能

#### 2.2.1 照明控制
- **亮度调节：** 0.1%-100% 无级调光，记忆上次亮度
- **色温调节：** 2700K (暖白) - 6500K (冷白) 连续可调
- **RGB彩光：** 1600万色，支持轮播、呼吸、爆闪
- **情景模式：**
  - 阅读模式 (4000K, 80%)
  - 聚会模式 (彩色轮播)
  - 睡眠模式 (2700K, 10%)
  - 应急模式 (高亮白光 100%)

#### 2.2.2 智能联动
- **时间控制：** 日出日落自动开关，定时任务
- **传感器联动：** 接入人体/光敏传感器（通过Matter）
- **语音控制：** "Hey Siri, 打开露营灯"
- **群组控制：** 同时控制多盏灯（最多16盏）

#### 2.2.3 能源管理
- **电量显示：** APP实时显示剩余电量（百分比+小时）
- **低功耗模式：** <10%电量自动进入超长续航（亮度限20%）
- **太阳能优化：** 晴天日照5小时充满，阴天续航维持

## 3. 用户角色与用户故事

### 3.1 用户角色

| 角色 | 描述 | 需求 |
|------|------|------|
| 露营者 | 户外过夜，营地照明 | 续航>48h，防水，易操控（戴手套） |
| 庭院家庭 | 花园装饰，夜晚氛围 | 美观，色彩丰富，定时开关 |
| 餐厅经营者 | 露台营业照明 | 商业级亮度，集中控制，耐用 |
| 公园管理员 | 公共区域照明 | 太阳能免维护，防破坏 |

### 3.2 用户故事

#### US-01: 作为露营者，我需要帐篷外照明持续整夜
- **验收：** 低亮模式（10%）下续航>48小时，IPX8防水应对降雨
- **优先级：** P0

#### US-02: 作为庭院用户，我想通过手机APP调节灯光颜色和场景
- **验收：** APP内有调色盘，10秒内完成RGB设置并生效
- **优先级：** P1

#### US-03: 作为餐厅老板，我可以统一控制所有露台灯的开关和亮度
- **验收：** 在APP内创建"露台"群组，一键打开5盏灯并设为阅读模式
- **优先级：** P1

#### US-04: 作为用户，我希望无需网关直连智能音箱
- **验收：** 支持Matter协议，Apple Home/Google Home 扫描添加成功
- **优先级：** P2
