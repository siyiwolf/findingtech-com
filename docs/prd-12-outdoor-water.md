## 1. 产品概述

### 1.1 产品名称
**产品代号：** PureWater Pro (PWP-300)  
**产品定位：** 户外场景下的高流量、多层过滤净水系统，支持细菌/病毒/化学污染物三重防护

### 1.2 产品简述
PureWater Pro 是一款泵式户外净水器，集成陶瓷 pre-filter、超滤膜 (0.01μm)、UV-C 杀菌灯和活性炭吸附四层过滤。目标流速 2L/min，适合 4-6 人团队快速净水。可选配重力袋或水袋兼容。适用于长途徒步、探险旅行、应急备用场景。

### 1.3 核心价值主张
- **三重防护：** 物理过滤 (细菌/寄生虫) + UV-C (病毒) + 活性炭 (化学/异味)
- **大流量：** 2L/min 泵式出水，4人团队无需等待
- **耐用长寿命：** 陶瓷滤芯 >50,000L，超滤膜 >2,000L，UV-LED >10,000小时
- **智能提醒：** 蓝牙APP记录滤芯寿命、UV灯状态、总处理水量

## 2. 功能规格

### 2.1 技术参数

| 参数 | 规格 | 备注 |
|------|------|------|
| **流量** | 1.5-2.5 L/min (取决于水源清洁度) | 泵式压柄操作 |
| **过滤精度** | Pre-filter: 1µm + UF: 0.01µm + UV-C: 99.99% | 三层 |
| **UV-C 波长** | 254nm，30秒杀灭病毒细菌 | LED 灯 |
| **活性炭** | 烧结炭块，吸附余氯/有机污染物 | 500L 寿命 |
| **进水口** | 复合 hose (1.5m)，带预滤网 | 防止大颗粒 |
| **出水口** | 螺纹接口，兼容标准水袋/水瓶 | |
| **材质** | Tritan 壳 + 陶瓷 + 不锈钢 | 食品级 |
| **重量** | ≤420g (主机) + 配件 80g | |
| **尺寸** | Ф85mm × H260mm (泵筒) | 紧凑 |

### 2.2 用户交互

- **泵柄：** 双动式（推拉双向出水），力度适中（儿童可使用）
- **流量指示：** 绿色/红色指示灯（流量>1.5L/min 绿灯，否则红灯）
- **UV 状态：** 蓝色 LED 闪烁表示工作，常亮表示故障
- **APP 连接：** 蓝牙 5.0，记录滤芯寿命（基于处理升数）

## 5. 技术需求

### 5.1 性能
- 最大压力: 6 bar
- UV-C 剂量: ≥40 mJ/cm² (确保病毒灭活)
- 滤芯安装: 快拆式（5秒更换）
- 温漂: -10°C ~ 50°C 性能不降级

### 5.2 认证
- NSF/ANSI 53 (健康效应) 和 58 (反渗透/UF) 或等效
- ISO 9001 生产过程
- 专利：陶瓷再生技术（清洗可恢复80%通量）

## 7. 风险

- 认证周期长 (NSF 6-9个月) → 提前送样，同步市场预热
