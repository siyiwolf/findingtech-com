## 1. 产品概述

### 1.1 产品名称
**产品代号：** SmartLink Fusion (SLF-500)  
**产品定位：** 4G+卫星双模便携Wi-Fi热点，专为数字游牧者、深度户外爱好者设计

### 1.2 产品简述
SLF-500 是一款支持4G Cat.6与低轨卫星（Starlink Mobile / 国产星链）智能切换的移动Wi-Fi设备。内置双卡（移动/联通/电信）+ 卫星模块，无4G信号时自动启用卫星。支持Wi-Fi 6E，10设备同时连接，续航20小时（4G模式）/ 5小时（卫星模式）。集成GPS/北斗定位和SOS一键呼救。

### 1.3 核心价值主张
- **双模永续联网：** 4G + 卫星智能切换，覆盖城市到荒野
- **续航焦虑解除：** 20小时续航 + 太阳能板（选配）可延长至48小时
- **资费透明：** 国内包天¥50/天，卫星套餐¥100/小时
- **安全定位：** 内置GPS/北斗，SOS一键发送位置短信

## 2. 功能规格

### 2.1 硬件规格

| 组件 | 规格 |
|------|------|
| 无线 | 4G Cat.6 (DL300Mbps/UL50Mbps) + Wi-Fi 6E (AX1800) |
| 卫星 | 集成低轨卫星模块（Starlink Mobile 或国产等效） |
| 定位 | GPS L1 + 北斗 B1，支持AGPS |
| 主控 | 高通 SDX65 (5G 基带) + IPQ5018 (Wi-Fi 6E) |
| 内存 | 4GB LPDDR4 + 64GB UFS |
| 电池 | 10000mAh 锂聚合物，支持PD 30W快充 |
| 显示 | 1.3" OLED (128×64) 显示信号/电量/号码 |
| 接口 | USB-C (充电+数据), nano-SIM×2, TF卡槽 |
| 防护 | IP54 (防尘防泼溅)，工作温度 -10°C~50°C |
| 重量 | 280g (主机) |
| 尺寸 | 120mm × 70mm × 18mm |

### 2.2 功能特性

- **双模切换：** 4G信号<5%时，<30秒自动切换卫星模式（用户无感）
- **多卡聚合：** 2张SIM卡绑定，自动选最优运营商
- **资费管理：** APP内套餐订购（中国移动/联通/电信 + 卫星），实时扣费
- **SOS紧急：** 侧边实体键，长按3秒发送GPS位置+求助短信（通过卫星）
- **设备共享：** 最多10台设备同时连接，带宽动态分配
- **离线地图：** 内置全球基础地图，无网也可查看位置轨迹
- **太阳能兼容：** 支持18-30V太阳能板直充（4A）

## 3. 用户故事

#### US-01: 作为数字游牧者，我希望在无4G区域保持网络连接
- 验收: 进入沙漠/深山，4G信号消失，30秒内卫星连接成功，可收发微信
- 优先级: P0

#### US-02: 作为户外博主，我需要稳定上传视频素材
- 验收: 卫星模式下上行速率≥5Mbps，1080p视频上传10分钟完成
- 优先级: P0

#### US-03: 作为徒步者，我希望设备轻便且续航持久
- 验收: 主机重量<300g，4G模式续航20小时，支持移动电源充电
- 优先级: P1

#### US-04: 作为用户，我担心资费天价
- 验收: APP内清晰显示当前速率¥/MB，达到¥50自动断连（可设置封顶）
- 优先级: P1

## 5. 技术需求

### 5.1 关键性能
- 4G下行峰值 ≥300Mbps (Cat.6)，实际续航 20h @ 5W
- 卫星模式上行 ≥5Mbps，续航 5h @ 15W
- 切换时间: <30s (从4G消失到卫星可用)
- 支持10设备并发，公平调度
- SAR值 < 1.6 W/kg (安全标准)

### 5.2 认证
- SRRC (无线电型号核准，中国) - 6个月
- 3C (中国) - 3个月
- FCC/CE (美/欧) - 4个月
- 卫星频段使用许可 (需要运营商合作)

## 6. 项目计划 (7个月，2,000台)

- Month 0-2: 硬件设计 + 卫星对接 (需运营商合作)
- Month 3: EVT原型5台 (4G+卫星双模验证)
- Month 4: DVT 50台，小流量测试
- Month 5: 认证送样 (SRRC/3C/FCC/CE并行)
- Month 6: PVT 200台，代工厂爬坡
- Month 7: 量产发布

**风险缓冲：** 卫星资费谈判可能延迟3个月，准备Plan B（北斗短报文）
