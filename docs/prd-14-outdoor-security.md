## 1. 产品概述

### 1.1 产品名称
**产品代号：** OutdoorSec Pro (OSP-100)  
**产品定位：** 针对露营、公园、社区、校园场景的智能化户外安防系统

### 1.2 产品简述
OutdoorSec Pro 是一款集成 AI 边缘计算、4G/5G 传输、太阳能供电的户外安防摄像头系统。产品设计用于无电源环境（如露营地、公园），支持全天候运行（IP66防尘防水），提供实时监控、移动侦测、人脸识别（云端可选）、本地存储（SD卡/NAS）功能。

### 1.3 核心价值主张
- **无电源部署：** 内置太阳能板 + 大容量锂电池，满足阴天3天续航
- **低功耗AI：** 边缘计算芯片，支持人形/车辆侦测，误报率<5%
- **多网融合：** 4G/5G + WiFi（营地热点）自动切换，保障传输稳定
- **隐私优先：** 支持本地存储，数据不出本地，可选云端加密备份

### 1.4 目标市场
- **B2C：** 高端露营者、庭院家庭（后院安防）
- **B2B：** 露营基地、公园管理处、中小企业园区
- **B2G：** 社区居委会、小型学校、应急救援点

### 2.2 软件功能

#### 2.2.1 核心AI功能
| 功能 | 描述 | 指标 |
|------|------|------|
| 人形识别 | 区分人/动物/车辆 | 召回率>95%，误报率<5% |
| 移动侦测 | 区域框选，灵敏度可调 | 响应时间<1s |
| 人脸识别 (可选) | 本地/云端比对，白名单 | 准确率>98% (云端) |
| 声音侦测 | 玻璃破碎、哭声、叫喊 | 灵敏度-40dB~80dB |
| 事件分类 | 标记"可疑"、"正常"、"报警" | AI自动打标 |

#### 2.2.2 管理端功能
- **移动APP** (iOS/Android)：
  - 实时预览（H.264/H.265）
  - 回放检索（按时间/事件类型）
  - 截图/录像下载
  - 告警推送（推送+邮件+短信）
  - 云端备份（加密，可选）
- **Web管理后台**：
  - 批量设备管理（适用于B2B客户）
  - 录像存储策略设置
  - 用户权限管理（三级：管理员/操作员/访客）
  - AI模型远程升级（OTA）

#### 2.2.3 安全与隐私
- 数据加密：AES-256本地加密，TLS 1.3传输加密
- GDPR合规：支持本地存储模式（数据不上云）
- 访问控制：双因素认证（2FA）+ IP白名单
- 日志审计：所有操作记录，支持导出

## 3. 用户角色与用户故事

### 3.1 用户角色

| 角色 | 描述 | 主要需求 |
|------|------|----------|
| **露营者** | 个人用户，露营地短住（3-7天） | 远程监控营地财物、安全告警、简单易用 |
| **营地管理者** | 企业用户，管理多个营地 | 集中管控多台设备、人员权限、事件归档 |
| **庭院家庭** | 家庭用户，后院安防 | 低成本、可扩展、隐私保护 |
| **公园管理员** | B2G用户，公共区域监控 | 耐候性高、太阳能供电、 vandal-proof |

### 3.2 用户故事（Epic拆解）

#### US-01: 作为露营者，我想远程查看营地实时画面，确保财物安全
- **验收标准：**
  - 打开APP，3秒内看到实时画面（4G信号）
  - 支持上下左右旋转云台（PTZ，选配）
  - 截图/录像一键保存到手机
- **优先级：** P0

#### US-02: 作为露营者，我希望有人入侵时收到手机推送
- **验收标准：**
  - 晚上10点后，检测到人形，APP推送+短信
  - 用户可设置"静默时段"
  - 推送包含缩略图和事件录像片段（10秒）
- **优先级：** P0

#### US-03: 作为营地管理者，我可以集中管理多台设备
- **验收标准：**
  - Web后台查看所有设备状态（在线/离线）
  - 批量设置录像策略（如"所有设备本地存储30天"）
  - 分配子账号给员工（不同权限）
- **优先级：** P1

#### US-04: 作为隐私敏感用户，我可以选择纯本地存储
- **验收标准：**
  - 设置中关闭"云端备份"
  - 所有录像仅存储在TF卡/NAS
  - AI识别在本地完成，无数据上传
- **优先级：** P1

#### US-05: 作为普通用户，安装不超过5分钟
- **验收标准：**
  - 开箱后，扫码下载APP
  - 按引导3步完成绑定（插电→配网→注册）
  - 第一次移动侦测设置不超过3次点击
- **优先级：** P2

## 8. 约束与假设

### 8.1 约束
- 必须支持4G Cat.6 和 WiFi 5 双模
- 必须满足 IP66 防护等级
- 必须通过 CCC、CE、FCC 认证（3C为强制）
- AI模型必须本地运行，云端可选

### 8.2 假设
- 太阳能板在阴天仍可提供 2-3 小时有效充电
- 4G 网络覆盖目标市场（中国主要城市可达，偏远地区降级）
- 用户对隐私敏感度中等，约 60% 选择本地存储
- 代工厂良率初期 80% → 6 个月内提升至 95%
