## 1. 产品概述

### 1.1 背景
户外露营、自驾游等场景中，物品收纳和管理是高频痛点。传统收纳箱功能单一，无法满足智能化、可视化管理需求。智能存储系统通过集成传感器、通信模块和智能算法，实现物品的自动识别、状态监测、位置追踪和空间优化，提升户外活动效率和体验。

### 1.2 目标
- 为用户提供**智能识别**（RFID/图像识别）和**状态监控**（温湿度、开关门）的存储解决方案。
- 实现**远程管理**（手机App控制）和**语音交互**（与智能音箱联动）。
- 支持**模块化扩展**（传感器、电池、通信模块可选）。
- 确保**户外耐用性**（防水IP67、耐-20℃低温）。

### 1.3 核心价值
| 价值主张 | 描述 |
|----------|------|
| 智能管理 | 自动识别箱内物品，支持列表化管理和搜索 |
| 状态感知 | 实时监测温湿度、门磁状态，异常告警 |
| 远程控制 | App远程查看、开锁、设置模式 |
| 模块化 | 可按需扩展传感器（如空气质量、重量） |
| 户外坚固 | IP67防水、抗冲击、宽温工作 |

### 1.4 目标用户
| 用户群体 | 场景 | 需求 |
|----------|------|------|
| 露营家庭 | 装备收纳、儿童玩具管理 | 分类清晰、耐用、易搬运 |
| 自驾游者 | 车辆空间有限，需快速取用 | 模块化堆叠、快速定位 |
| 户外俱乐部 | 批量装备管理，租借跟踪 | 批量识别、使用记录 |
| 家庭储物间 | 季节性物品管理（如节日装饰） | 防潮、防尘、长期保存 |

### 1.5 使用场景
- **露营收纳**：多层级收纳箱，识别不同类目（炊具、衣物、工具），App显示清单。
- **车载运输**：堆叠设计，固定带，防滑底座；行驶中状态监测。
- **户外库房**：多箱联网，统一管理，出入库记录。
- **家庭后院**：长期存放园艺工具、玩具，防潮防霉。

## 2. 用户故事与用例

### 2.1 用户故事
| 角色 | 需求 | 价值 |
|------|------|------|
| 露营妈妈 | 快速找到孩子的换洗衣物 | 节省时间，减少翻找 |
| 自驾游爱好者 | 确认后备箱所有装备齐全 | 避免遗漏，安心出行 |
| 俱乐部管理员 | 跟踪30个收纳箱的位置和内容 | 提高管理效率，减少丢失 |
| 家庭用户 | 监测存储间温湿度，防止物品变质 | 保护贵重物品 |

### 2.2 用例分析
1. **物品注册**：用户放置物品 → 按下“添加”按钮 → 系统语音提示“请说出物品名称” → 用户说出“登山杖” → 系统记录到该箱清单。
2. **智能盘点**：打开App → 选择“XX收纳箱” → 显示当前箱内物品列表（名称、数量、上次使用时间）。
3. **异常告警**：箱门意外打开 → 系统发送通知到手机。
4. **环境监测**：湿度超过70% → App推送提醒，建议除湿。
5. **语音查询**：对智能音箱说“收纳箱里有多少个手电筒？” → 语音回复“ currently 3个”。

## 3. 功能规格

### 3.1 智能识别模块
- **RFID方案**（主）：每个箱体配备RFID读写器，每件物品粘贴无源RFID标签（防水）。
- **图像识别**（可选）：内置广角摄像头 + 轻量AI模型，识别常见物品（支持物品分类训练）。
- **语音录入**：麦克风 + 语音识别（离线关键词），用于快速添加物品名称。
- **二维码辅助**：打印二维码贴纸，用户扫码快速注册物品。

### 3.2 环境监测模块
- **温湿度传感器**：Sensirion SHT45，精度 ±2%RH / ±0.2℃
- **门磁传感器**：干簧管，检测箱门开关状态
- **空气质量**（可选）：VOC传感器，检测有害气体
- **光照传感器**：检测箱内是否被打开（通过光线变化）

### 3.3 通信与连接
- **主通信**：蓝牙 5.2（低功耗，连接手机App）
- **Mesh组网**（可选）：支持多个收纳箱之间通过 Thread/Zigbee 组网，扩展覆盖
- **云端同步**：Wi-Fi 桥接（通过手机或独立网关）上传数据到云端
- **本地存储**：MicroSD 卡槽（最大 256GB），离线记录

### 3.4 电源与续航
- **电池**：可充电锂离子电池 18650 × 2 (3000mAh)，支持户外太阳能板充电（5V/2A）
- **续航**：正常监测模式（每小时上报） ≥ 6个月；频繁使用（RFID扫描） ≥ 2个月
- **低功耗设计**：待机电流 < 50μA，门磁唤醒 < 1秒
- **充电接口**：USB-C（支持 PD 18W）

### 3.5 硬件规格
- **外壳材料**：ABS + PC 复合，IP67 防水防尘，MIL-STD-810H 耐冲击
- **尺寸**：大号 60L (600×400×300mm)，中号 40L (500×350×280mm)，小号 20L (400×300×250mm)
- **重量**：大号 2.5kg（含电池），中号 1.8kg，小号 1.2kg
- **显示屏**（可选）：1.3" OLED 屏，显示箱内物品数量、温湿度、电量
- **按键**：电源、模式、录音（添加物品）
- **指示灯**：RGB LED 显示状态（充电、连接、告警）

### 3.6 软件功能
- **App (iOS/Android)**：
  - 设备管理（添加/删除收纳箱）
  - 物品清单查看、搜索、编辑
  - 温湿度曲线、历史告警
  - 固件OTA升级
  - 多用户共享（家庭共享）
- **语音助手集成**：支持天猫精灵、小爱同学、Siri Shortcuts（语音查询）
- **云端服务**（可选）：数据备份、跨设备同步、AI推荐（物品归类建议）

## 4. 技术架构

### 4.1 系统架构
```
+-------------------+
|   用户交互层       | ← App / 语音 / 按键
+-------------------+
|   业务逻辑层       | ← 物品管理、告警、同步
+-------------------+
|   硬件抽象层       | ← RFID/传感器/通信驱动
+-------------------+
|   电源管理         | ← 电池、充电、功耗控制
+-------------------+
```

### 4.2 核心组件选型
| 组件 | 型号/规格 | 供应商 | 备注 |
|------|-----------|--------|------|
| MCU | ESP32-H2 (Wi-Fi 6 + BLE 5.2) | Espressif | 主控，低功耗 |
| RFID读写器 | RC522 (13.56MHz) | NXP | 读卡距离 ≤5cm |
| 温湿度 | SHT45 | Sensirion | I2C接口 |
| 电池 | 18650 3000mAh ×2 | ATL | 可充电 |
| 外壳 | PC+ABS 合金 | 定制 | IP67密封 |
| 天线 | PCB板载天线 | 自研 | 优化射频性能 |

### 4.3 数据流
- **物品注册**：用户触发 → MCU录制语音/读取RFID → 云端保存物品-标签映射 → App同步。
- **盘点查询**：用户请求 → MCU扫描箱内所有标签 → 返回ID列表 → 云端查询名称 → App显示。
- **告警推送**：传感器事件（门磁、湿度） → MCU通过BLE/Wi-Fi → 云端推送至App。

### 4.4 安全与隐私
- 通信加密：BLE pairing + TLS (云端)
- 数据本地存储可选（不上传）
- 支持匿名模式（不绑定用户身份）

## 5. MVP 范围

### 5.1 MVP功能清单
- 基础存储箱（40L，含RFID读写器、温湿度、蓝牙）
- iOS/Android App（设备绑定、物品清单、温湿度显示）
- 语音添加物品（离线关键词）
- 门磁告警推送
- USB-C充电，续航 ≥ 3个月

### 5.2 排除项
- 图像识别（v2.0）
- Mesh组网（v2.0）
- OLED显示屏（可选配件）
- 云端AI推荐（后续版本）

### 5.3 验证指标
- RFID识别准确率 ≥ 99.5%（10cm内）
- 蓝牙连接稳定性 ≥ 99%（10米内）
- 续航时间 ≥ 90天（正常监测模式）
- App响应延迟 < 200ms（本地操作）
- 防水等级 IP67（30分钟 1米水深）
