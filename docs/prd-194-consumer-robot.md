# PRD-194: Consumer Robot (个人消费级机器人)

## 1. 概述

**产品名称**: Finding Buddy (暂定)  
**目标市场**: 个人消费者（都市家庭、科技爱好者）  
**产品形态**: 轮式移动机器人，可选配机械臂模块  
**核心功能**: 自主清洁、家庭安防、语音陪伴、轻量级物品搬运

## 2. 目标用户

| 用户类型 | 特征 | 使用场景 |
|---------|------|----------|
| 都市双职工 | 25-45岁，收入$80k+ | 自动化清洁、home security |
| 科技爱好者 | 18-35岁，早期采用者 | 新奇功能体验、API开放 |
| 老年独居 | 65+，需要监护 | 紧急呼叫、日常陪伴 |

## 3. 功能需求

### 3.1 核心功能 (MVP)

- **自主导航**: 激光SLAM + 视觉融合，避开障碍物
- **基础清洁**: 扫 + 拖，尘盒自动集尘 (基站)
- **安防巡视**: 定时巡逻，移动侦测报警
- **语音交互**: 内置麦克风阵列，支持语音控制（"Hey Buddy"）
- **App控制**: iOS/Android App，远程控制、地图编辑

### 3.2 进阶功能 (V2)

- **机械臂模块**: 5DOF，抓取1kg以内物品（水杯、遥控器）
- **多模态AI**: 大模型对话（情感陪伴、知识问答）
- **多机协同**: 多台机器人协同作业（分区清洁）

### 3.3 性能指标

- 续航: 4小时连续工作，快充1小时至80%
- 噪音: 清洁模式 ≤60 dB
- 越障: 可爬1.5cm门槛
- 防摔: 楼梯跌落检测 + 自动后退
- 精度: 定位误差 ≤2cm

## 4. 非功能需求

- **安全**: UL 60335-1/2-10, FCC Part 15, CE RED
- **隐私**: 数据本地处理（语音、图像），可选云端分析
- **可靠性**: MTBF ≥5,000小时
- **兼容性**: WiFi 5/6, Bluetooth 5.2

## 9. 风险与缓解

- **技术风险**: SLAM在复杂场景失败 → 多传感器冗余 + 云端fallback
- **供应链风险**: 激光雷达缺货 → 锁定2家供应商（速腾+禾赛）
- **安全风险**: 机械臂伤人 → 力检测 + 急停开关
- **隐私风险**: 视频数据泄露 → 端侧处理 + E2E加密
