## 1. 产品概述

### 1.1 产品定义

CampBrew 户外咖啡系统是一套集研磨、萃取、温控、自动冲洗于一体的便携式全自动咖啡机，专为精致露营、户外旅行场景设计。支持一键制作意式浓缩、美式、手冲等多种咖啡，内置电池续航 50 杯，满足 3-5 人小团队需求。

### 1.2 目标市场

- **精致露营爱好者**：周末短途露营，追求高品质生活
- **户外旅行博主/摄影师**：需要高颜值、便携设备
- **民宿/露营地**：提供增值咖啡服务
- **户外婚礼/活动**：移动咖啡站

### 1.3 核心价值主张

| 价值点 | 说明 | 量化指标 |
|--------|------|----------|
| 极致便携 | 一体化设计，无需携带多个器具 | 重量 <2.5kg，体积≈A4纸 |
| 操作极简 | 一键完成从豆到杯 | 3步操作（加水加豆→选择→按压） |
| 品质稳定 | PID 温控 + 预浸泡，专业级萃取 | 水温精度 ±1°C，压力 9bar |
| 超长续航 | 内置大电池，支持多次制作 | 50杯（畅享版）/ 或外接移动电源 |
| 快速清洁 | 自动冲洗 + 可拆卸，户外友好 | 3分钟完成清洁 |

### 1.4 定价

- **基础版**：¥1,299（手压萃取 + 锥刀磨豆 + 温控，需外接水源）
- **畅享版**：¥2,499（内置电池 + 自动上水 + 自动冲洗 + App 控制）
- **配件**：便携咖啡杯套装 ¥199，咖啡豆订阅 ¥99/月

## 3. 功能规格

### 3.1 基础版（CampBrew Basic）

**功能**：
- 手压手动萃取（无需电池）
- 锥刀磨豆机（35档可调）
- PID 温度控制（88-96°C 三档预设）
- 可拆卸水箱（500ml）
- 压力表可视化

**性能**：
- 萃取时间：25-30秒
- 出品量：30-60ml（单份浓缩）
- 重量：2.2kg
- 尺寸：200×150×300mm

### 3.2 畅享版（CampBrew Pro）

**新增功能**：
- 内置 20000mAh 电池，支持 50 杯连续制作
- 自动上水系统（可选配 1.5L 外置水箱）
- 自动冲洗（一键清洗）
- OLED 显示屏（参数、剩余杯数）
- App 连接（配方自定义、温度曲线、使用统计）
- 电动泵压（一键萃取，无需手动）

**性能**：
- 充电时间：4小时（Type-C 65W）
- 连续制作：50杯（每杯30ml浓缩）
- 重量：2.5kg
- 尺寸：220×160×320mm

## 4. 技术架构

### 4.1 系统框图

```
┌────────────────────────────────────────────┐
│          CampBrew Pro (畅享版)              │
├────────────────────────────────────────────┤
│  水箱 → 磨豆 → 粉碗 → 加热冲泡 → 流出      │
│         │        │        │              │
│    锥刀研磨   PID控温   9bar泵压         │
│         │        │        │              │
│    控制板 ← OLED + 按钮 + 蓝牙            │
│              │                            │
│           锂电池 20000mAh                 │
└────────────────────────────────────────────┘
```

### 4.2 软件逻辑

**萃取流程**：
1. 加水至水箱（自动检测水位）
2. 加咖啡豆（自动称重，建议 18g）
3. 选择咖啡类型（浓缩/美式/手冲）
4. 点击开始（预热 → 研磨 → 压粉 → 萃取 → 流出）
5. 完成提示，可选自动冲洗

**App 功能**（畅享版）：
- 配方自定义：研磨度、水温、压力曲线
- 使用统计：制作杯数、豆水比、清洗提醒
- 固件OTA升级

## 5. 用户体验

### 5.1 开箱即用

1.  unpack，检查配件（主机、手柄、粉碗、清洁刷）
2.  首次使用：清水冲洗水箱、空运行一次
3.  开始制作：加豆、加水、按键

### 5.2 清洁维护

- 每使用 5 次：自动冲洗（1分钟）
- 每日：倒弃粉饼，冲洗粉碗
- 每月：深度清洁（除垢剂）

## 6. MVP范围 (v1.0)

### 6.1 包含功能

✅ **基础版**：
- 手压萃取（9bar）
- 锥刀磨豆（35档）
- PID 温控（3档）
- 压力表
- 无电池，需外接水源（自带水箱也可）

✅ **畅享版**（基础版全部 +）：
- 内置电池 + 自动上水
- 自动冲洗
- OLED 屏 + App 控制
- 电动萃取（一键）

### 6.2 排除功能（v2.0+）

- ❌ 奶泡系统（太重，不符合户外轻量化）
- ❌ 双锅炉（成本高，体积大）
- ❌ 磨豆机校准提示（高级功能）
- ❌ 离线地图/导航（与咖啡无关）

## 7. 非功能性需求

| 需求 | 指标 |
|------|------|
| 重量 | 基础版 ≤ 2.2kg，畅享版 ≤ 2.5kg |
| 尺寸 | 不超过 A4 纸包裝 |
| 噪音 | 磨豆 ≤ 65dB（户外可接受） |
| 续航 | 畅享版 ≥ 50 杯（单次充电） |
| 温度范围 | 操作环境 -10°C ~ 40°C |
| 防护等级 | IP54（防尘防泼溅） |
| 萃取温度精度 | ±1°C |
| 萃取压力稳定性 | ±0.5bar |
