## 1. 产品概述

### 1.1 产品定义

RideHub 是一款集成导航、骑行数据记录、智能照明、语音控制、摄影云台于一体的自行车智能中枢设备。通过夹持式设计适配各类自行车，内置电池续航 20 小时，支持 GPS/北斗双模定位，旨在替代手机和多个单功能配件，提升骑行安全与便利。

### 1.2 目标市场

- **城市通勤骑行者**：导航、安全照明、防盜
- **运动骑行爱好者**：数据记录（速度、踏频、功率）、训练分析
- **长途冒险骑手**：装备管理、紧急SOS、续航保障
- **骑行内容创作者**：稳定云台拍摄、语音控制

### 1.3 核心价值主张

| 价值点 | 说明 | 量化指标 |
|--------|------|----------|
| 一体化 | 替代手机+码表+灯+云台+传感器 | 5合1，节省重量 <1kg |
| 安全提升 | 智能灯光 + SOS + 导航语音提示 | 夜间事故率降低预期 30% |
| 数据驱动 | 多维度骑行数据分析 | 速度、踏频、功率、心率、路线 |
| 拍摄便利 | 语音控制云台，免手持 | 拍摄效率提升 5 倍 |
| 超长续航 | 20 小时 + 太阳能辅助 | 跨日长途无焦虑 |

### 1.4 定价

- **基础版**：¥1,599（导航 + 码表 + 智能灯 + SOS + 蓝牙）
- **Pro版**：¥2,999（增加 3轴云台 + 功率计 + 踏频传感器 + 功率仪）
- **配件**：太阳能充电板 ¥299，手机App免费，云存储 ¥9.9/月

## 3. 功能规格

### 3.1 基础版功能

- **导航与码表**：
  - 离线地图预装（中国主要城市）
  - 语音提示转向、红绿灯
  - 实时显示：速度、距离、时间、海拔、坡度
- **智能照明**：
  - 自动感光开关
  - 刹车检测（减速自动高亮）
  - 转向灯（可选配）
- **安全SOS**：
  - 一键呼救（短信+位置发送给紧急联系人）
  - 摔倒检测（加速度判断，自动警报）
- **蓝牙连接**：
  - 同步手机通知（来电、消息）
  - 导出数据到 Strava/Keep

### 3.2 Pro 版额外功能

- **摄影云台**：
  - 手机/运动相机 3轴稳定
  - 语音控制："拍照"、"录像"、"跟踪我"
  - 自动追踪骑行手势
- **功率训练**：
  - 功率计集成（左右脚踏）
  - 踏频/速度/功率曲线分析
  - 训练计划推送（通过App）
- **心率带支持**：
  - 蓝牙连接 ANT+ 心率带
  - 实时心率区间提示

## 4. 技术架构

### 4.1 系统框图

```
┌─────────────────────────────────────────────┐
│            RideHub 主控单元                  │
│  导航 + 显示 + 存储 + 通信 + 电源           │
├─────────────────────────────────────────────┤
│     传感器接口：速度、踏频、功率、心率       │
├─────────────────────────────────────────────┤
│     云台模块（Pro） + 电机驱动               │
├─────────────────────────────────────────────┤
│     蓝牙 5.2 + 可选 4G                      │
└─────────────────────────────────────────────┘
```

### 4.2 软件功能

- **本地固件**：FreeRTOS，核心任务：定位、显示、传感器采集、云台控制
- **手机App（RideHub）**：路线规划、视频剪辑、数据分析、OTA升级
- **云服务**：用户账号、轨迹存储、训练计划、社交分享

## 5. 用户体验

### 5.1 安装

- 快拆夹具：适配 22-31mm 圆形管、方管
- 3分钟完成安装（无需工具）
- 线缆隐藏式走线

### 5.2 日常使用

- 开机即用：自动定位，显示当前速度/心率
- 语音播报：转弯提示、超速警告、电量提醒
- 一键SOS：长按3秒触发，确认防误触
