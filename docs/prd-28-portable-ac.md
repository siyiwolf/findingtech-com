# 28-便携空调 - PRD

**版本**: v1.0  
**日期**: 2026-04-18  

---

## 1. 概述

**CoolTent** 便携式变频空调，专为 5-10m³ 空间（帐篷、小房车、户外工棚）设计。支持市电或内置电池供电，低噪音模式≤35dB，10分钟降低 10°C。Pro 版支持 4-6h 电池续航。

---

## 2. 核心功能

- 变频压缩机 R290（环保冷媒）
- 制冷量 800 BTU (标准版) / 1200 BTU (Pro)
- 能效比 >3.0
- 噪音: 标准 42dB / 静音 35dB
- 电池: 2000Wh 磷酸铁锂 (Pro)
- 重量: 标准 9kg / Pro 12kg (含电池)
- 冷媒充注: R290 30g
- 控制: 面板 + 红外遥控 + App
