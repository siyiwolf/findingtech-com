# 30-充气桑拿 - PRD

**版本**: v1.0  
**日期**: 2026-04-18  

---

## 1. 概述

**AirSauna** 充气式便携桑拿房，1-4人容量，20分钟升温至 80°C，适合庭院、露营、康复场景。Pro 带 App 控制+香薰+负离子。

---

## 2. 核心功能

- 充气结构: TPU 双层，电动气泵 3 分钟完成
- 加热: 1.5kW PTC，温度范围 40-90°C
- 均匀性: 循环风扇，温差 <5°C
- 控制: 旋钮（基础）/ App（Pro）
- 附加: 香薰盒（Pro）、负离子发生器（Pro）
- 尺寸: 展开 180×120×200cm，收纳 50×30cm
- 重量: 8kg

## 5. KPI

- 升温时间 ≤ 20min (到80°C)
- 温差 ≤ 5°C
- 耐久性: 500 次充放气
- 面料耐温 ≥ 90°C
