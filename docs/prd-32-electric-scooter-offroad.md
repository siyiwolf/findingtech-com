# 32-越野电动滑板车 - PRD

**版本**: v1.0  

---

## 1. 概述

**AllRoad** 可折叠越野电动滑板车，双电机 2×500W，10英寸真空胎，双液压避震，可拆卸电池续航 50-60km，IP65 防水。

---

## 2. 核心功能

- 动力: 双电机 500W×2，峰值 80Nm，爬坡 25°
- 轮胎: 10英寸真空胎，越野花纹
- 避震: 前液压 + 后弹簧
- 折叠: 3秒折叠，尺寸 <50cm，重量 17kg
- 电池: 48V 15Ah 可拆卸，续航 50-60km
- 速度: 限速 25km/h（符合国标），可解限35
- 控制: LCD 仪表 + 蓝牙 App
- 灯光: 前 LED 大灯 + 刹车灯

---

## 3. 产品组合

- 城市版 ¥3,299: 单电机 350W, 8.5英寸胎, 续航 30km
- 越野版 ¥5,999: 双电机 500W, 10英寸胎, 双避震, 50km
