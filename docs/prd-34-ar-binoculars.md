# 34-AR望远镜 - PRD

**版本**: v1.0  

---

## 1. 概述

**StarSight** AR 望远镜，集成小型目镜显示器、图像识别芯片、自动对焦电机，提供观测目标名称和信息叠加，支持拍照录制。

---

## 2. 核心功能

- 光学: 10×42  Roof 棱镜，视场 6.5°
- AR 显示: 目镜内嵌微型 OLED, 叠加信息
- 识别: 内置 NPU，离线识别鸟/星/地标
- 对焦: 电机驱动，点击对焦目标
- 拍照: 8MP CMOS，保存至 TF 卡 + App
- 数据库: 鸟类 2000种 + 星座 88 + 地标 5000
- 电池: 可更换 18650，续航 4h

---

## 3. 产品线

- 观鸟版 ¥2,999: 鸟库 + 自动对焦 + 拍照
- 观星版 ¥3,999: + 星库 + 赤道仪追踪 + AR 标注
- 教育版 ¥2,499: 简化 UI + 课件

---

✅ AR 显示

## 5. KPI
