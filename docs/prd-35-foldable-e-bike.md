# 35-折叠电单车 - PRD

**版本**: v1.0  

---

## 1. 概述

**CityFold** 轻量电动折叠自行车，符合新国标，10秒快折，续航 50-60km，重量 ≤ 18kg，适合地铁接驳+城市通勤。

---

## 2. 核心功能

- 电机: 36V 350W 无刷中置电机
- 电池: 36V 10Ah 可拆卸锂电，续航 50-60km
- 重量: ≤ 18kg (标准) / ≤ 16kg (高配)
- 折叠: 10 秒单手握持，尺寸 <80cm
- 变速: 7 速 Shimano
- 仪表: LCD 显示速度、电量、里程
- 连接: Bluetooth App 里程/故障诊断
- 灯光: 前 LED 大灯 + 尾灯

---

## 3. 产品组合

- 标准版 ¥3,299: 铝合金车架，18kg，50km
- 高配版 ¥4,599: 碳纤维前叉+轮组，16kg，60km

---

## 4. MVP

✅ 轻量折叠 ≤ 18kg  
✅ 续航 ≥ 50km  
✅ 快折 10s  
✅ 电池可拆带锁  

---

## 5. KPI

- 折叠时间 ≤ 12s
- 续航（25km/h 平路）≥ 50km
- 重量合格率 ≥ 90%
- 电池循环 500次 ≥ 80%
