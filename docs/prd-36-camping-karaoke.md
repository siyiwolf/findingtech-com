# 36-露营卡拉OK - PRD

**版本**: v1.0

## 1. 概述

**CampSing** 户外便携 KTV 系统，集成主音箱、2 支 UHF 麦克风、低音炮（Pro），防水 IP65，内置电池续航 8h，海量云端曲库。

---

## 2. 核心功能

- 音频: 2×50W RMS + 低音炮 100W（Pro）
- 麦克风: UHF 双麦克风，自动配对
- 音效: DSP 混响 + 电音 + 原唱切换 + AI评分
- 曲库: 云端 200 万+，每周更新，会员无限
- 连接: 蓝牙 5.2 + WiFi + AUX
- 防水: IP65（主箱 + 麦克风）
- 电池: 内置 20Ah，6-8h（中等音量）
- 控制: 面板 + 遥控 + App
