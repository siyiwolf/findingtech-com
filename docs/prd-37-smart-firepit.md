# 37-智能火坑 - PRD

**版本**: v1.0  

---

## 1. 概述

**FlameHub** 智能酒精火坑，自动供料燃烧，无烟可控，安全围栏，App/遥控调节。

---

## 2. 核心功能

- 燃料: 生物乙醇 (95%)，5L 内置油箱，续航 8-12h
- 燃烧: 专用燃烧室，火焰自然跳动，热输出 5-15kW 可调
- 控制: 电子阀调节火力，App/遥控/手动
- 安全: 防倾倒自动断料，高温围栏
- 烟雾: 设计低烟，顶棚导流可选
- 香薰: 可选配香薰盒（松木等）
- 尺寸: 直径 80cm，高 60cm
- 材质: 不锈钢 + 耐火砖

---

## 3. 产品组合

- 标准版 ¥2,999: 5L油箱 + 遥控 + 围栏
- Pro版 ¥4,999: + App + 风道调烟 + 香薰
