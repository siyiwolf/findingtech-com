# 38-驱蚊机器人 - PRD

**版本**: v1.0  

---

## 1. 概述

**MosquitoBot** 自动巡航驱蚊机器人，UV光诱+微雾+CO2模拟三重防护，适合庭院、营地。

---

## 2. 核心功能

- 驱蚊: UV 385nm 诱捕 + 氯氟醚菊酯微雾 + CO2 模拟
- 移动: 轮式底盘，随机/边界巡航，200m² 覆盖
- 感应: 人体红外 + CO2 浓度，靠近自动加强
- 续航: 内置 10Ah 锂电，8h，支持太阳能
- 药液: 100ml 可拆卸，低毒
- 噪音: ≤ 30dB
- 防护: IP44
