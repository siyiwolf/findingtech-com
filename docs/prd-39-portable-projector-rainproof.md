# 39-防水投影 - PRD

**版本**: v1.0  

---

## 1. 概述

**RainShow** 户外防水便携投影，IP65，800 ANSI 流明，4-5h 续航，自动对焦校正，内置 Android TV。

---

## 2. 核心功能

- 光学: DLP 0.23", 800 ANSI 流明, 1080p
- 防水: IP65 (暴雨级)
- 续航: 内置 20000mAh，4-5h @ 中等亮度
- 校正: 自动对焦 + 四向梯形 ±30°
- 系统: Android TV 11, 支持 App
- 音响: 2×10W 立体声，蓝牙 5.2
- 接口: HDMI, USB-C (PD 输入)
- 重量: 1.8kg

---

## 3. 产品组合

- 户外版 ¥2,999: IP65 + 800流明 + 4h 续航
- 旗舰版 ¥3,999: + 自动校正 + Android TV + 双10W
