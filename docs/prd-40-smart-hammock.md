# 40-智能吊床 - PRD

**版本**: v1.0  

---

## 1. 概述

**AutoHang** 智能电动吊床，一键收紧/松开，角度自动调节，承重监控，电动收起。

---

## 2. 核心功能

- 绑绳: 两端电机收绳，适应 2-4m 树距
- 角度: 电机调节两端高度差，平躺/半躺
- 监控: 压力传感器，超重报警
- 收起: 电动卷绳，5分钟完成
- 控制: 面板 + 遥控 + App
- 材质: 高强尼龙 70D，承重 200kg
- 电池: 可拆卸 5000mAh，支持 50 次操作

---

## 3. 产品组合

- 标准版 ¥1,999: 收紧+角度+承重监控
- Pro版 ¥2,999: + 电动收起 + App + 遥控器

---

## 4. MVP

✅ 电机收紧/松开  
✅ 角度调节  
✅ 承重监控  
✅ 电动收起（Pro）

---

## 5. KPI

- 收紧时间 ≤ 30s
- 承重准确率 ±5kg
- 续航 ≥ 50 次循环
- 整体承重 ≥ 200kg
