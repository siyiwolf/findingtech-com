# 41-泳池清洁机器人 - PRD

**版本**: v1.0  

---

## 1. 概述

**PoolBot** 自动泳池清洁机器人，智能路径规划，强力吸附爬墙，长续航，WiFi 控制。

---

## 2. 核心功能

- 导航: 陀螺仪+编码器，逐行清洁，防缠绕
- 吸附: 底部旋转刷 + 吸力 ≥ 2000Pa
- 续航: 锂电 15000mAh，3-4h
- 覆盖: 150m² / 次
- 过滤: 双层滤网（大垃圾 + 微尘）
- 回收: 定时返回岸边挂钩
- 连接: WiFi 预约控制

---

## 3. 产品组合

- 标准版 ¥2,499: 基础路径 + 3h 续航
- Pro版 ¥3,999: + 爬墙能力 + 自动回收 + WiFi

---

## 4. MVP

✅ 智能路径  
✅ 强力吸污  
✅ 3h 续航  
✅ 过滤系统  

---

## 5. KPI

- 清洁覆盖率 ≥ 95%
- 爬坡角度 ≥ 30°
- 续航 ≥ 3h
- 过滤精度 ≤ 100μm
