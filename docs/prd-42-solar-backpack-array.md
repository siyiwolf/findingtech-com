# 42-折叠光伏阵列 - PRD

**版本**: v1.0  

---

## 1. 概述

**SunFold** 可快速展开的大功率折叠太阳能板，200W/500W 可选，内置 MPPT，适配便携储能或房车充电。

---

## 2. 核心功能

- 功率: 单板 200W (标准) / 500W (Pro)
- 电池: 无（输出到外部储能）
- 展开: 一体化支架，磁吸连接，3 分钟
- 重量: 200W 2.8kg; 500W 5.2kg
- 输出: 12V/5A, 24V/3A, USB-C PD 100W (Pro)
- 控制器: 内置 MPPT，效率 >97%
- 防护: IP65，可小雨工作
