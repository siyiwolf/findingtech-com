# 45-便携制冰机 - PRD

**版本**: v1.0  

---

## 1. 概述

**IceGo** 便携快速制冰机，6-10 分钟出冰，日产量 12kg，支持多水源，多电源选项。

---

## 2. 核心功能

- 制冰: 压缩机循环，子弹冰，日产量 12kg
- 速度: 首颗冰 6-10 分钟
- 储冰: 内置保温层，冰块 6h 不化
- 水源: 自来水/纯净水/瓶装水（内置过滤）
- 电源: 12V 车充 (10A) / 220V 家用 / 户外电源 PD 200W
- 噪音: 基础 50dB，静音版 45dB
- 重量: 4.8kg

---

## 3. 产品组合

- 基础版 ¥1,299: 6分钟制冰，噪音 50dB
- 静音版 ¥1,999: 10分钟，噪音 45dB + 过滤

---

## 4. MVP

✅ 快速制冰  
✅ 多电源兼容  
✅ 便携轻量  
✅ 保温储冰  

---

## 5. KPI

- 首冰时间 ≤ 10min
- 日产量 ≥ 12kg
- 噪音达标 (版本)
- 保温时间 ≥ 6h
