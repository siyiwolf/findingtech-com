# 46-生态移动厕所 - PRD

**版本**: v1.0  

---

## 1. 概述

**EcoLoo** 可移动堆肥厕所，好氧微生物分解，无污水排放，智能监控，适合大型活动与露营地。

---

## 2. 核心功能

- 堆肥: 好氧发酵，减量 90%，出肥料
- 通风: 负压风机 + 活性炭，无异味
- 监控: 温度、湿度、满载传感器，App 提醒
- 翻堆: 电机定期搅拌
- 模块: 单蹲位 + 主机处理单元
- 能源: 市电 220V 或 200W 太阳能
- 材质: 玻璃钢耐腐蚀

---

## 3. 产品组合

- 单间套装 ¥8,999 (含主机 + 1蹲位)
- 扩展蹲位 ¥3,000/个

---

## 4. MVP

✅ 好氧堆肥处理  
✅ 通风除臭  
✅ 智能监控  
✅ 模块化

---

## 5. KPI

- 日处理量 ≥ 50 人次/蹲位
- 异味等级 ≤ 2
- 满载预警提前 24h
- 耗电 ≤ 1kWh/天
