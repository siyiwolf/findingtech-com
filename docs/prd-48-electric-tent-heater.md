# 48-帐篷电暖器 - PRD

**版本**: v1.0  

---

## 1. 概述

**TentWarm** PTC 陶瓷加热器，安全无明火，静音，适合帐篷小空间保暖。标准版 220V，电池版 12V。

---

## 2. 核心功能

- 加热: PTC 陶瓷，300W / 600W 两档
- 安全: 过热 + 倾倒自动断电
- 噪音: ≤ 30dB
- 控制: 机械旋钮
- 电源: 标准版 220V; 电池版 12V 10A
- 电池版续航: 6000mAh，3h (600W)
- 重量: 1.2kg
