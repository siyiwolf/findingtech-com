# 49-户外WiFi路由器 - PRD

**版本**: v1.0  

---

## 1. 概述

**WildNet** 户外多网聚合路由器，4G/5G + 可选卫星，IP67，长续航，支持 50+ 终端，提供稳定户外网络。

---

## 2. 核心功能

- 网络: 4G Cat-20 (标准) / 5G NSA (Pro)
- WiFi: Wi-Fi 6 (802.11ax), 覆盖半径 200m (户外)
- 聚合: 多链路绑定，提升带宽与可靠性
- 续航: 20000mAh 电池，12h 正常负载
- 充电: 12V/2A，可选 50W 太阳能板
- 防护: IP67，-30~60°C
- 管理: Web + App，流量统计、黑名单

---

## 3. 产品组合

- 标准版 ¥1,299: 4G + WiFi6 + IP67
- Pro版 ¥2,499: + 5G + 太阳能 + 50+ 终端

---

## 4. MVP

✅ 4G 接入 + WiFi6  
✅ IP67 防护  
✅ 12h 续航  
✅ 50 终端并发

---

## 5. KPI

- 下载速度 ≥ 1Gbps (Pro)
- 续航 ≥ 12h
- 温度范围 -30~60°C
- 带机量 ≥ 50
