# 50-智能遮阳帽 - PRD

**版本**: v1.0  

---

## 1. 概述

**CoolCap** 智能户外帽，集成半导体制冷、UV 监测、通风风扇、反光安全设计。

---

## 2. 核心功能

- 制冷: TEC 片，头顶降温 5-10°C（Pro）
- 通风: 微型风扇，循环头部空气
- UV: 传感器，实时显示强度，App 提醒
- 反光: 夜间高可见度条
- 电源: 可拆卸 1000mAh，续航 8h（中等功率）
- 控制: 按钮/App 开关、档位
- 可水洗: 电子模块快拆

---

## 3. 产品组合

- 标准版 ¥199: 风扇 + 通风
- 智 cold 版 ¥399: + TEC + UV 传感器

---

## 4. MVP

✅ 通风 + 风扇  
✅ UV 监测（Pro）  
✅ 制冷（Pro）  
✅ 反光安全

---

## 5. KPI

- 制冷温差 ≥ 5°C
- 续航 ≥ 8h
- 重量 ≤ 150g
- UV 精度 ±1 级
