1. 市场机会概述

1.1 全球仓储机器人市场

全球仓储机器人市场规模2023年约$8.2B，预计2030年达到$35B，CAGR 23.1%（MarketsandMarkets, 2024）。电商爆发、劳动力短缺、效率需求三大驱动力。

分拣机器人是其中增长最快的细分，2023-2030年CAGR 27.4%，规模将从$2.1B扩张至$12.8B。

1.2 中国市场

中国物流机器人市场2023年约¥5.5B（$780M），预计2023-2030年CAGR 32.8%，2030年突破¥35B（中国物流与采购联合会，2024）。电商巨头（阿里、京东、拼多多）持续投入自动化，制造业升级（智能制造2025）推动需求。

1.3 我们的切入点

现有玩家：瑞士Swisslog、美国GreyOrange、海康机器人、快仓。差距：高端被海外垄断，国产低价但性能弱。机会在于：高性能国产替代 + 灵活模块化设计 + AI智能调度。

2. 用户画像与核心痛点

2.1 典型客户

客户类型规模应用场景采购特点
电商仓储中心日单量>50,000订单分拣、退货处理追求效率、准确性、快速部署
制造业零部件仓SKU 10,000+物料配送、产线供给需要与WMS/MES系统集成，可靠性高
第三方物流（3PL）多客户、多品类混合订单分拣灵活性、可配置、快速切换
零售门店仓城市配送中心门店补货订单紧凑设计、噪音低、易操作

2.2 核心痛点

  1. 部署周期长：传统AGV项目3-6个月，影响旺季备货
  2. 灵活性差：Layout变更需重新编程，难以应对促销峰值
  3. 成本高：进口机器人单价¥200,000+，国产便宜但故障率高
  4. 系统封闭：难以与现有WMS系统对接，数据孤岛
  5. 维护复杂：需要专业工程师，响应慢

3.2 竞争空白

  • 轻量级部署：中小规模（10-50台）用户需要快速方案，现有商业版太重，开源太费时。
  • 国产自主可控：政府、国企、关键基础设施偏好国产，避免制裁风险。
  • AI原生调度：传统基于规则，缺乏学习优化能力。
4. 产品概念与差异化定位

4.1 产品名称：SortBot 集群分拣系统

Slogan: "快速部署，智能调度，极致灵活"

4.2 系统架构

  • SortCore 边缘服务器：运行编队调度引擎，支持1000+机器人协同，响应时间<50ms
  • SortAgent 机器人端软件：轻量级客户端（C++），负责定位、避障、通信
  • SortCloud 云端平台：远程监控、数据分析、OTA升级、数字孪生
  • SortBridge 设备桥接：适配主流机器人底盘（AGV/AMR）、传感器（激光雷达、摄像头）
  • SortStudio 配置工具：图形化拖拽式场景编辑，一键部署

4.3 核心能力

能力规格价值
快速部署导入地图 → 设置任务 → 下载到机器人，30分钟完成避免数周调试，快速见效
AI调度深度强化学习优化路径，拥堵减少40%，吞吐提升25%效率提升，成本降低
弹性扩展线性扩展到500台，性能平滑下降<5%支持业务增长
高可用边缘服务器双机热备，单点故障不影响运行7x24可靠
国产自主不依赖ROS（兼容），核心算法自研，通过等保三级满足政策与安全要求

4.4 产品线

  • SortBot Pro：支持10-100台机器人，高级AI调度，¥300,000/年起
  • SortBot Standard：支持10-50台，基础调度，¥120,000/年起
  • SortCloud SaaS：按机器人数量订阅，¥1,000/台/月，含OTA、数据分析
