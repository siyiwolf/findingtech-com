版本: v2026-04-18-insight
分类: 户外餐饮
负责人: MKT Agent

1. 机遇

  • 火候难控: 传统炭火温度波动大，新手易烤焦
  • 翻面麻烦: 需 Manual 翻动，人多时忙不过来
  • 油滴起火风险: 危险

2. 客群

  • 家庭后院烧烤
  • 露营爱好者
  • 烧烤新手

3. KSF

  • 智能温控: PID 保持设定温度 ±5°C
  • 自动翻面: 电机旋转串签，均匀受热
  • 防起火: 油接盘 + 水冷降温
  • App 监控: 温度曲线、烹饪建议

4. 定价

- 标准版（温控）: ¥899 - Pro（+自动翻面）: ¥1,599
