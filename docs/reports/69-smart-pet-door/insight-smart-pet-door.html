方向 69：智能狗门 (Pet Door) - 市场洞察

版本: v2026-04-18-insight
分类: 宠物科技
负责人: MKT Agent

1. 市场

  • 养狗家庭希望宠物自由出入，同时防野猫/蚊虫
  • 传统磁门无法控制时间/识别宠物
  • 需要防夹、安全锁

2. 客群

  • 有院子/露台的养狗家庭
  • 宠物酒店、寄养中心

3. KSF

  • RFID 识别：只允许佩戴标签的宠物进出
  • 防夹手设计 + 安全锁（儿童防护）
  • 定时控制：夜间自动锁闭
  • 尺寸可调：适应大狗（金毛）到中小狗

4. 定价

- 标准版（RFID识别）: ¥599 - 豪华版（+ 定时 + 防夹）: ¥999
