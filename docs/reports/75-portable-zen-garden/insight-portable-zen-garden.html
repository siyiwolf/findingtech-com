方向 75：便携禅园 (Portable Zen Garden) - 市场洞察

1. 市场

  • 冥想、办公桌减压需求
  • 传统沙盘需 Manual 耙，不够智能
  • 可结合AR、声音增强体验

2. 客群

  • 白领工作者
  • 冥想爱好者
  • 礼品市场

3. KSF

  • 自动耙: 小型电机 + 磁吸耙，可编程图案
  • 电子沙: 变色LED底光，营造氛围
  • App: 选择图案、速度、背景音乐（自然声）
  • 便携: 尺寸 30×30cm，重量 1.5kg，电池续航 8h

4. 定价

- 基础版（手动耙+LED）: ¥299 - 自动版（电机+App）: ¥699
