方向 87：便携街机 (Portable Arcade Machine) - 市场洞察

1. 市场

  • 怀旧游戏风潮，玩家希望随时随地玩经典街机
  • 现有掌机（Switch）侧重新游戏，缺街机摇杆体验

2. 客群

  • 80/90后怀旧玩家
  • 通勤族、旅行者
  • 电竞酒店、露营娱乐

3. KSF

  • 7寸IPS屏，720p
  • 内置千款经典街机游戏（模拟器）
  • 可拆卸摇杆+按键（支持多人对战）
  • 电池续航 6-8h
  • HDMI 输出到大屏

4. 定价

- 标准版 (预装1000款): ¥399 - 豪华版 (可扩展 + 手柄) : ¥599
