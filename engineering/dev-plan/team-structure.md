## 📝 角色定义

### 硬件工程组
- **机械工程师**：3D 设计（SolidWorks）、DFM、样品打样、量产跟进
- **电子工程师**：原理图、PCB（Altium）、元器件选型、EDA 仿真
- **测试工程师**：环境测试（高低温、跌落）、EMC、可靠性

### 嵌入式软件组
- **嵌入式主管**：技术路线、代码审查、团队管理
- **底层驱动工程师**：MCU（STM32）驱动、RTOS、低功耗
- **ROS 2 工程师**：导航、机械臂控制、多机协同
- **安全工程师**：安全启动、加密、固件签名、渗透测试

### 后端服务组
- **后端主管**：微服务架构、API 规范、性能调优
- **微服务工程师**：Go/Python 开发、数据库设计、消息队列
- **数据工程师**：ETL、数据仓库、BI 报表
- **SRE**：K8s 运维、监控、CI/CD 流水线维护

### 前端与移动组
- **前端主管**：前端架构、组件库、性能优化
- **Web 工程师**：Vue3/TypeScript、地图可视化、图表
- **Flutter 工程师**：跨平台 App、蓝牙、卫星通信 SDK
- **UI/UX 设计师**：交互原型、视觉设计、设计系统

### 系统架构组
- **首席架构师**：整体技术架构、技术选型、对外技术交流
- **算法工程师**：调度算法（RL）、风险预警模型、轨迹预测
- **DevOps 工程师**：云基础设施、K8s、GitOps、成本优化

## 🔄 协作机制

### 跨组协作
- **硬件 + 嵌入式**：硬件接口协议定义，联调测试
- **嵌入式 + 后端**：通信协议（API + MQTT），数据格式
- **后端 + 前端**：API 契约先行（OpenAPI），Mock Server
- **算法 + 后端**：模型部署格式，推理服务接口
- **全员 + 架构**：架构评审（设计阶段）+ 代码评审（开发阶段）

### 沟通工具
- **项目管理**：Jira / GitHub Projects
- **文档**：GitLab Wiki / Notion
- **即时通讯**：企业微信 / 钉钉 / Slack
- **代码**：GitLab CE（自建）或 GitHub Enterprise
- **设计**：Figma（UI/UX 协作）
