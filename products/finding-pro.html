功能详解

🛰️

双模卫星通信

集成铱星9603模块和北斗短报文模块。在无手机信号区域，可拨打/接听卫星电话、发送/接收短信、一键SOS全球救援。

铱星覆盖全球，北斗覆盖中国及周边。卫星套餐需单独订阅。

🧠

端侧AI引擎

内置专用AI芯片，5 TOPS算力。本地运行环境感知模型，实时分析气压趋势、运动模式、声音事件，无需上传云端即可发出预警。

保护隐私，响应更快，节省带宽。

🗺️

专业离线导航

支持导入多种格式离线地图（GeoTIFF、MBTiles等）。1.5英寸OLED屏幕阳光下清晰可见。Turn-by-turn语音导航，兼容GPX/KML路线。

可加载全球任意区域地图。

📡

多星定位系统

同时接收GPS、北斗、GLONASS、Galileo卫星信号。城市峡谷、密林等复杂环境定位更快更准，精度可达亚米级。

最高支持L1+L5双频。

🌡️

环境传感器套件

高精度气压计、温度/湿度传感器、指南针、陀螺仪。支持连接外部气象站、水质仪、土壤探头等科研设备。

通过USB-C或蓝牙连接。

💪

超强续航与耐用

2800mAh电池，连续使用72小时。支持USB-C PD快充，2小时充满80%。IP68防护，-40°C至85°C工作，1.5米跌落无损。

极端环境电池化学配方。

🔄

LoRa 自组网

内置LoRa模块，设备间可直接通信（无基站）。多名队员组队时，即使卫星无信号也能共享位置、发送文字。

最远10公里（视距）。

🔗

开放 API

提供完整的REST API和SDK（Python、C、MATLAB）。可集成到自己的系统，自动化数据下载、设备配置、告警订阅。

支持Webhook、MQTT。
